Durch die Konstruktion mit window.onload wird sichergestellt, dass das Fenster vollständig geladen wurde, bevor der Code ausgeführt wird.

Prinzipiell wird das Skript jetzt ausgeführt, wenn auch alle eingebundenen Ressourcen, z.B. Bilder geladen worden sind. In der Praxis gibt es daher noch andere Methoden, weil man zum Beispiel nicht auf diese Ressourcen warten möchte, bis der Code ausgeführt wird.

Wie gesagt, man kann das script-Tag auch einfach weit hinten im Code platzieren. So, oder so, innerhalb dieses window.onload können wir erstmal experimentieren.
